Project-PORTRAIT

portrait curve

Something which started as a fun curious little activity in early 2019 is now grown into something substantial. If it wasn't for random tutorial of drawing digital portrait on my youtube homepage back then, all this wouldn't have been possible.

Started on Adobe Draw, which is a beginner friendly software for anyone looking to get into digital art, all the initial work was made on this. Soon enough moved to Adobe Skecth for more features and intricacy, then finally settled on Adobe Fresco which has all the features one might need.
